# CareerVault

A powerful career development and resource management application designed to help professionals organize, track, and advance their career goals effectively.

## 📋 Project Overview

CareerVault is a modern web application built to serve as a centralized platform for career management. Whether you're job hunting, skill building, or planning your professional growth, CareerVault provides all the tools you need in one intuitive interface.

## ✨ Features

- **Career Resource Organization** - Centralize all your career-related documents, links, and resources
- **Goal Tracking** - Set and monitor professional goals with progress tracking
- **Profile Management** - Maintain a comprehensive professional profile
- **Resource Library** - Build a personal library of career development materials
- **User-Friendly Interface** - Intuitive design for seamless navigation
- **Responsive Design** - Works perfectly on desktop and mobile devices

## 🚀 Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required for basic usage

### Installation

1. Clone the repository
```bash
git clone https://github.com/mittals-shubham/CareerVault.git
cd CareerVault
```

2. Open the project
```bash
# Simply open index.html in your browser
# Or use a local server for better development experience
```

3. Using a local server (recommended)
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js with http-server
npx http-server
```

4. Access the application
```
Open your browser and navigate to http://localhost:8000
```
